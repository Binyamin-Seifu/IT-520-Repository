# Lab 01 Assets

This folder contains proof of work, evidence, and supporting documentation for Lab 01: Enterprise Infrastructure Audit.

## Contents:

- Network topology diagrams
- BGP configuration analysis
- Screenshot evidence of analysis
- Log files and command outputs
- Network packet captures

## Usage:

All assets referenced in Lab-01.md should be stored here with descriptive filenames:

- `network-topology-before.png` - Pre-outage infrastructure diagram
- `bgp-route-analysis.png` - BGP routing table analysis
- `dns-query-output.txt` - DNS resolution attempts during outage
- `recovery-timeline.png` - Visual timeline of incident and recovery
